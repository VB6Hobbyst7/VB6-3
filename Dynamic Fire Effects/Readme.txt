 DESCRIPTION:
-----------------------------------------------------------------------------
This is a demonstration of real-time fire in Direct3D 8. Code presents 4 
different high-resolution environments using fire effects: fire pit, candle, 
streetlamp, torch, furn, campfire, lamp, and chandelier.

* Requires a graphics card with pixel shader 1.4 or higher
